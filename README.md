# cardscript-doc-generator

> Produces markdown documentation for Cardscript.

## <a name="install"></a>Install
```bash
$ npm install cardscript-doc-generator --save
```

## <a name="usage"></a>Usage
```bash
$ npm run generate
```

## <a name="test"></a>Testing

```bash
$ npm test
```
